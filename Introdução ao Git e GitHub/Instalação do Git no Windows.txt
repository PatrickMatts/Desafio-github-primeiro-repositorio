Após baixar o Git em https://gitforwindows.org/, siga estes passos:

Instale o Git com as configurações padrões. 

Abre o Git Bash e configure seu nome de usuário e endereço de e-mail com os seguintes comandos: 

git config --global user.name "Seu Nome" 
git config --global user.email "nome@email.com" 

Agora, o Git está pronto para uso.
